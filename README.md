Nagios Plugins
==============

Three types of files:
* \*.sh : The actual script (Bash, here)
  * To put in /usr/local/nagios/libexec
* \*.php: The Pnp4Nagios template
  * To put in /usr/local/pnp4nagios/templates
* \*.png: A sample graph output
